## Installation

Application requires [PHP] >= 7.1 to run.

Install the dependencies and devDependencies and start the server.

```sh
git clone https://github.com/raulidavid/claroinsurance.git
cd claroinsurance
composer install
php artisan serve
go to (http://localhost:8000)
```
